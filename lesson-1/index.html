Git команди:

git config --global user.name - команда, що встановлює ім`я користувача в системі Git;

git config --global user.email - команда, що встановлює адресу електронної пошти користувача в системі Git;

git init - команда, що створює новий підкаталог .git, який містить всі необхідні файли репозиторія;

git status - команда, що показує стан файлів( які файли змінились, контрольовані та неконтрольовані Gitом файли, в якій гілці знаходимось і т.д.)

git add - команада, яка вказує Git що файл потрібно відслідковувати;

git commit -?????????

git log - команда, що виведе зроблені коміти репозиторія;
